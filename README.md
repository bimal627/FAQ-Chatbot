# AI FAQ Chatbot

## 📌 Description
An AI-powered FAQ Chatbot developed using Python and Streamlit as part of the CodeAlpha Internship Program. The chatbot answers frequently asked questions through a simple and interactive interface using Natural Language Processing (NLP) techniques.

## ✨ Features
- Answers frequently asked questions
- User-friendly Streamlit interface
- Fast and accurate response generation
- Uses TF-IDF Vectorizer and Cosine Similarity
- Easy to customize with new FAQs

## 🛠️ Technologies Used
- Python
- Streamlit
- scikit-learn
- NumPy
- VS Code
- Git
- GitHub

## ▶️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/bimal827/CodeAlpha-Task2-FAQ-Chatbot.git
```

### 2. Install the required libraries

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
streamlit run app.py
```

- Streamlit
